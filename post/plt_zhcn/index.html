matplotlib中文绘图

September 13, 2021 · 1 min

在默认情况下，中文在plt中会被绘制成▯，猜测应该是因为默认字体不包含中文；

可以通过配置字体的方法支持中文，大部分博客给出的字体是SimHei（黑体），但是渲染出来并不美观，经过我的测试，个人最推荐SimSun（宋体）。

from matplotlib import pyplot as plt
'''👇两行配置'''
plt.rcParams['font.sans-serif'] = ['SimSun'] # 修改字体为宋体
plt.rcParams['axes.unicode_minus'] = False	# 正常显示 '-'
